What are javascript values ?
The javascript syntax defines two types of values:
	1.fixed values
	2.variable values
		fixed values are called literals.
		variable values are called variables.
--
What are rules for javascript Literals?
	The two important rules for fixed literals are
		1. numbers are written with or without decimals.
		Example:
		1000
		10.50
	2.Strings  text written within double or single quotes.
		Example:
		"john"
		'john'
--
What are javascript variables?
	 In a programming  language, variables are used to store data values.
	 Javascript uses keywords var, let and const to declare variables.
	 An equal sign (=) is used to assign values to variables.
		Example:
		let x;
		x = 6;

there are two types of operators.

javascript uses an assignment operator(=) to assign values to variables.
