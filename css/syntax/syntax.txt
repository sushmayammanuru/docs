 How to write a css code?
	A CSS consists of a selector and a declaration block.
	where selector points to the html element you want to style.
	Declaration block contains one or more declarations,
	separated by semicolon;
	Each declaration includes  a css property-name and property-value,
	separated by a colon[:].
	Declaration block is sorrounded by curly braces.
--
Syntax:
selector {
	property-name1:property-value1;
	property-name2:property-value2;
	property-name3:property-value3;
	}
--
Example:
h3{
	color:red;
	background-color:green;
	border:1px solid red;
	}
--
