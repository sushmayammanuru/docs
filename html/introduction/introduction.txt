What is an HTML element?
	HTML element is defined by start tag,some content,end tag.
	- -
Syntax:
		<tag name>some content here</tag-name>
Example:	
<h1> content here one</h1>
<h6>content here six</h6>

	some elements have no content.

	empty elements have no end tag.

	<br>,<img>,<link>,<input>..etc

what is the difference between tag and elements?
	A tag may be a start tag or end tag,it doesnot have content.
	example:
	<h1>,<p>,</h1>,</p>...etc.
	An element is a combination of <start tag>,some content and </endtag>
	Example:
	<h6> content here six</h6>
