What is favicon?
	A favicon is a small image displayed left to the page title in the browser.
	tab.
--
How to add favicon?
	you can use any image like as your favicon icon.
	you can also create your own favicon on sites like https://www.favicon.cc
	--
	To add a favicon to your website,either save your favicon image to the
	root directory of your webserver,or create a folder in the root directory
	called images,and save your favicon image in this folder a common name
	for a favicon image is "favicon.ico".
	Example:
	<head>
		<link rel="icon" type="image/x-icon" href="/images/favicon.ico">
		
	</head>
--

what is the tag we have to use to show favicon icon in the browser?
	link tag
What is the attribute we have to use for favicon? src or href or both?
	href attribute.
