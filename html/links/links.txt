HTML link_Hyperlinks
	HTML links are Hyperlinks
	you can click on a link and jump to another document.
	when you move the mouse over a link ,the mouse arrow will change to little
	hand.
--
HTML Links -the target attribute
	By default ,the linked page will displayed in the current browser window
	To change this,you must specify another target for the link.
	The target attribute specifies where to open the link.
	The target attribute can have one of the following values.
			_self
			_blank
			_parent
			_top
		Use target ="_blank" to open the linked document in a new browser tab.
		Example
			<a href="https://www.w3schools.com target="_blank">visit w3schools<a/>
use target="_self" to open the linked document in the current browser tab.
	Example:
	<a href ="https://www.w3schools.com/" target="_self">visit w3 schools</a>
--
LInk to an email address
	Use mailto:inside the href attribute to create a link that opens the users
	email program
	Example:
	<a href="mailto:someone mail @example .com">send mail</a>
--
Link Titles
	The title attribute specifies extra information about an element,
	the information most often shown an a tooltip text when the mouse moves
	over the element.
Example:
	<a href="https://www.w3schools.com/html" title="go to w3schools HTML section">w3schools</a>
