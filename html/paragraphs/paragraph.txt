HTML paragraph
	<p> element defines a paragraph.
	paragraph always starts on a new line and browsers automatically add some
	white space before and after a paragraph.
Example:
	<p>This is a first paragraph</p>
	<p>This is a second paragraph.</p>
--
Paragraph display
	you cannot be sure how paragraph will be displays.
	large or small screens ,resized windows will create default results.
	with html,you cannot change the display by adding extra spaces or extra
	lines in html code.
	The browser will automatically remove extra spaces or lines when the
	page is displayed.
Example:
	<p>This is a
	paragraph with
	many lines in
	source code</p>
	
	<p>This is a paragragph contains lot 
	of spaces in the source code
	but the browser ignores it</p>
	
	<p>The number of lines in the paragragph depends on the size of the 
	browser,if you resize  the browser window the number of linews in the
	paragragph.will change. </p>

Horizantal Rule
	The <hr> tag defines a thematic break in html page.
	It is most often  displayed in a horizantal rule.
	It is used to separate the content in an html page.
	The <hr>tag has no end tag,it is an empty element.
Example:
<h1>this a heading1</h1>
<p> this is a para 1</p>
<hr>
<h2> this is heading 2</h2>
<p>this is a para 2>/p>

Break line	
	the <br>element defines a break line.
	Use <br> if you want to break the without starting a new paragragph.
	The <br>tag is an empty tag,that means it has no end tag.
Example:
<p>this is a <br> paragragph with break line</p>
Poem problem
	The poem will display on a single line,if you written in <P>element.
Example:
	<p>
		My Bonnie lies over the ocean.
		My Bonnie lies over the sea.
		My Bonnie lies over the ocean.
		Oh, bring back my bonnie to me.
	</p>
