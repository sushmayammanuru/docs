What is HTML?
HTML stands for hyper text markup language.
It is the most building block of the web.
It is a tag based programming language.
It is the only language to create a webpage.
It describes the structure of the webpage.
It consist of many tags/elements(approx 100).
elements tell the browser how to display the content.
